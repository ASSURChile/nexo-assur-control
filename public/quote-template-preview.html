Maqueta para cotizaciones automáticas

Referencia visual y funcional para convertir las cotizaciones actuales en plantillas editables dentro de ASSUR Chile. La idea es que el sistema complete datos desde cliente, instalación, cotización, paquete, materiales, servicio recurrente y condiciones comerciales, permitiendo modificar cualquier texto y ajustar tablas antes de exportar.

Plantillas iniciales

Monitoreo de alarmas Sistemas de alarma, notas de compatibilidad y servicio mensual UF. Recurrente
Cerco eléctrico Metros lineales, valor de proyecto, equipamiento base y notas. Proyecto
Campos editables
IdentidadLogo, razón social emisora, datos empresa.
ContenidoIntroducción, notas técnicas, notas comerciales, cierre.
Datos automáticosCliente, fecha, productos, precio, MRR, forma de pago.

Constructor dentro de ASSUR Chile

Desde una cotización o propuesta, el usuario elige plantilla, revisa textos, ajusta condiciones y genera PDF.
